## Sobre o Projeto

Site institucional da Vinheria Agnello, loja de vinhos finos. Desenvolvido em HTML com Tailwind CSS.

## Páginas

- **Início** (`inicio.html`) - Página principal com coleção de vinhos
- **Vinícolas** (`vinicolas.html`) - Terroirs e propriedades
- **Nossa História** (`nossa-historia.html`) - Sobre a empresa
- **Assinatura** (`assinatura.html`) - Planos de membership
- **Carrinho** (`carrinho.html`) - Carrinho de compras
